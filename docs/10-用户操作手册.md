## 1. PC端（管理后台）

### 1.1 登录系统

**操作步骤：**

1. 打开浏览器，访问系统地址（如：http://localhost:5174）
2. 在登录页面输入用户名和密码
3. 输入验证码
4. 点击"登录"按钮

**注意事项：**
- 默认管理员账号：admin / 123456（首次登录后请立即修改密码）
- 验证码不区分大小写
- 连续5次登录失败将锁定账号15分钟

### 1.3 用户管理

**功能说明：**
管理系统用户账号，包括新增、编辑、删除、重置密码等操作。

**操作流程：**

#### 1.3.1 查看用户列表

1. 点击左侧菜单"用户管理"
2. 系统显示用户列表，包含：工号、姓名、性别、部门、职位、手机、状态等信息
3. 支持按部门、状态筛选
4. 支持按姓名、工号搜索

#### 1.3.2 新增用户

1. 点击"+ 新增用户"按钮
2. 在弹窗中填写用户信息：
   - 姓名（必填）
   - 性别（必填）
   - 手机号（必填）
   - 部门（必填）
   - 职位（必填）
   - 入职日期
3. 点击"保存"按钮
4. 系统自动生成工号和初始密码

#### 1.3.3 编辑用户

1. 在用户列表中点击"编辑"按钮
2. 修改用户信息
3. 点击"保存"按钮

#### 1.3.4 重置密码

1. 在用户列表中点击"重置密码"按钮
2. 确认重置操作
3. 密码重置为默认密码（123456）

#### 1.3.5 删除用户

1. 在用户列表中点击"删除"按钮
2. 确认删除操作
3. 用户被标记为删除状态

### 1.5 审批管理

**功能说明：**
处理各类审批申请，支持请假、报销、出差、加班等多种审批类型。

#### 1.5.1 查看待审批列表

1. 点击左侧菜单"审批管理"
2. 默认显示"待审批"标签页
3. 显示所有待处理的审批申请

#### 1.5.2 审批处理

1. 点击"审批"按钮
2. 查看审批详情：
   - 审批单号
   - 申请人信息
   - 申请类型和标题
   - 申请事由
   - 审批流程节点
3. 填写审批意见（可选）
4. 点击"同意"或"拒绝"按钮

#### 1.5.3 发起审批

1. 点击"+ 发起审批"按钮
2. 选择审批类型（请假/报销/出差/加班/采购/其他）
3. 填写审批标题和申请事由
4. 点击"提交申请"按钮

#### 1.5.4 撤回审批

1. 在"我发起的"标签页找到待审批的申请
2. 点击"撤回"按钮
3. 填写撤回原因
4. 确认撤回操作

## 3. 常见问题

### 3.1 登录问题

**Q：忘记密码怎么办？**
A：请联系系统管理员重置密码。

**Q：账号被锁定怎么办？**
A：账号连续5次登录失败会被锁定15分钟，请等待或联系管理员解锁。

### 3.2 考勤问题

**Q：打卡时提示"不在考勤范围内"怎么办？**
A：请确认是否在公司附近，或连接公司WiFi后重试。

**Q：忘记打卡怎么办？**
A：可以提交考勤申诉，说明原因后等待审批。

### 3.3 审批问题

**Q：审批被拒绝后可以重新申请吗？**
A：可以，修改申请内容后重新提交。

**Q：如何撤回已提交的审批？**
A：在"我发起的"标签页找到待审批的申请，点击"撤回"按钮。
