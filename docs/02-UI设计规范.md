## 一、设计原则

1. **一致性：** 全局使用统一的颜色、字体、间距、圆角、阴影规范
2. **层次感：** 通过字号、字重、颜色深浅区分信息层级
3. **简洁性：** 无多余装饰，信息密度适中，留白合理
4. **可操作性：** 交互元素有明确的 hover/active/focus 状态反馈
5. **响应式：** 侧边栏可折叠，表格可横向滚动，Modal 自适应宽度

## 三、字体系统

### 3.1 字体族

```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif;
```

### 3.2 字号层级

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 超大数字 | `28px` | `700` | 仪表盘统计卡片数值 |
| 大标题 | `22px` | `600` | 登录页系统名称 |
| 页面标题 | `18px` | `600` | 页面主标题 |
| 卡片标题 | `16px` | `600` | Modal 标题 |
| 小标题 | `15px` | `600` | 卡片内标题 |
| 正文 | `14px` | `400/500` | 表格内容、表单输入 |
| 次要正文 | `13px` | `400/500` | 按钮文字、标签、筛选项、表单 label |
| 辅助文字 | `12px` | `400` | 状态标签、描述文字 |
| 小型文字 | `11px` | `400` | 时间戳 |
| 最小文字 | `10px` | `400` | 菜单分组标题、Tab 徽标 |

### 3.3 使用规则

- **标题层级：** 页面标题 `18px/600` > 卡片标题 `15px/600` > Modal 标题 `16px/600`
- **正文层级：** 表格内容 `14px` > 按钮文字 `13px` > 状态标签 `12px` > 时间戳 `11px`
- **颜色层级：** 标题 `#1f2937` > 正文 `#374151` > 辅助 `#6b7280` > 占位 `#9ca3af`
