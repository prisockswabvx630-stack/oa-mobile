# 智能OA办公系统 - 项目概述

## 1. 项目背景

本项目旨在构建一套企业级智能OA（Office Automation）协同办公系统，覆盖企业日常办公的全场景需求，包括组织管理、审批流程、考勤打卡、任务协作、人事合同、薪资管理、资产管理、公告通知等核心业务。

系统采用前后端分离架构，由三端组成：

| 端 | 目录 | 技术栈 | 端口 |
|----|------|--------|------|
| 管理后台前端 | `admin` | Vue 3 + TypeScript + Vite 8 + 手写CSS | 5174 |
| 移动端App | `expo-domes` | React Native 0.83 + Expo SDK 55 + expo-router | 8081 |
| 后端API服务 | `oa_server` | Node.js + Express 5 + Prisma ORM + MySQL | 3000 |

## 2. 核心理念

- **RBAC权限模型**：基于角色的访问控制，支持菜单级、按钮级、API级精细化权限管理
- **数据权限隔离**：按部门及层级进行多维度数据权限过滤（全部/自定义/本部门/本部门及以下/仅本人）
- **多端一致性**：Web管理端与移动端业务逻辑高度一致，数据实时同步
- **工作流引擎**：内置BPMN 2.0规范的轻量级审批引擎，支持多级审批、会签、转办、加签

## 3. 功能架构

系统由七大板块构成：

```
┌─────────────────────────────────────────────────────┐
│                  用户端 (Web / Mobile)                │
├─────────────────────────────────────────────────────┤
│                  API网关 / 鉴权拦截器                  │
├──────────┬──────────┬──────────┬──────────┬─────────┤
│ 核心系统  │ 智能监控  │ 工作流   │ 协同办公  │ 消息中心 │
│ 管理     │ 中心     │ 引擎     │ 业务     │         │
├──────────┴──────────┴──────────┴──────────┴─────────┤
│              Prisma ORM + MySQL 8.0                  │
└─────────────────────────────────────────────────────┘
```

### 3.1 核心系统管理 (SYS)
用户管理、角色管理、菜单资源管理、部门架构、岗位管理、数据字典、系统配置、审计日志、公告通知、数据回收站

### 3.2 系统智能监控 (MONITOR)
在线用户管理、服务器负载监控、缓存监控、定时调度任务

### 3.3 工作流引擎 (WORKFLOW)
流程模板设计、审批链路控制、待办/已办/我发起的、驳回/转办/加签/会签、超时代理审批

### 3.4 智能考勤 (ATTENDANCE)
多端打卡（GPS/WiFi/设备校验）、考勤规则配置、异常申诉、假期联动、排班管理、月度报表

### 3.5 协同办公业务 (OA-BUSINESS)
人事档案、劳动合同、薪资工资条、资产管理、云盘文档、会议室预订、工作交接、费用报销、任务项目、企业用车、公文流转

### 3.6 全局消息中心 (MESSAGE)
系统消息、审批消息、业务消息、互动消息，多端同步，多渠道推送

### 3.7 开发辅助工具 (TOOLS)
代码生成器、交互式API文档

## 4. 技术选型总览

| 层级 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 移动端框架 | React Native (Expo) | SDK 55, RN 0.83 | 跨平台移动App |
| 移动端路由 | expo-router | - | 基于文件系统的路由 |
| 移动端API | fetch封装 | - | src/api/client.ts 统一封装 |
| 管理后台框架 | Vue 3 + Vite | Vue 3.5, Vite 8 | Composition API |
| 管理后台UI | 手写CSS | - | 无第三方UI组件库 |
| 管理后台路由 | v-if页面切换 | - | 无vue-router，App.vue中currentPage控制 |
| 管理后台状态 | Vue reactive | - | 原生响应式对象，无Pinia |
| 后端运行时 | Node.js | 18+ | JavaScript运行环境 |
| Web框架 | Express | 5.x | Web应用框架 |
| ORM | Prisma | 5.x | 新一代Node.js ORM |
| 数据库 | MySQL | 5.7+ (推荐8.0) | 关系型数据库 |
| 认证方式 | x-user-id请求头 | - | 前端传用户ID，后端信任（待升级JWT） |
